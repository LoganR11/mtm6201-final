My process:

My process with coding this website was very challenging. I have leanred over this semester that while coding can be fun when it goes smooothly, but when you are to code a full website, it takes lots of steps, focus, and precision. I did unfortunately learn that I much prefer designing websites rather than coding them, as this was a very tedious process. In the end, I think I actually almost nailed that part, but the designs and interactions is where I think that fell off a little.
